GUIDA RENDER - Deploy server (passo passo)
------------------------------------------
1) Crea account su https://render.com (consiglio Google Sign Up).
2) Crea repository GitHub con la cartella 'server' o carica direttamente lo zip nella UI.
3) Nuovo -> Web Service -> connect repository -> seleziona branch.
4) Imposta:
   Environment: python
   Build command: pip install -r requirements.txt
   Start command: uvicorn main:app --host 0.0.0.0 --port $PORT
5) Deploy. Copia l'URL del servizio e incollalo in app/config.json -> server_url
6) Per aggiornare edge map: visita GET /simulate_edge_map o carica edge_lookup.json aggiornato via repo.
