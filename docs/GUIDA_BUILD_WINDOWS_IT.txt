GUIDA SEMPLICE - Build & Test locale (Windows)
----------------------------------------------
Prerequisiti:
- PC Windows con Python 3.10+ installato.
- (Opzionale) Android Studio se vuoi usare emulator o buildare APK localmente.
- Git (consigliato).

Passi rapidi:
1) Estrai lo ZIP sul PC.
2) Apri terminale (PowerShell) nella cartella server:
   cd server
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   uvicorn main:app --reload --port 8000
3) Apri app/config.json e imposta "server_url" su http://localhost:8000
4) Esegui la demo Kivy sul PC (richiede Kivy). Per test rapido puoi leggere gli output del server usando Postman:
   POST http://localhost:8000/suggest_move
   payload sample in docs/ENGINE_USAGE_IT.txt

Se vuoi buildare APK localmente, installa Buildozer (Linux) o usa Android Studio (Windows) seguendo la guida nel file BUILD_INSTRUCTIONS.txt.
